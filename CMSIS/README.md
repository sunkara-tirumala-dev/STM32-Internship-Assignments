# STM32 CMSIS Programming

##  Description
This project demonstrates the use of CMSIS for peripheral programming in STM32F401RE.

##  Objective
To simplify register-level programming using CMSIS standard definitions.

##  Features
- Easy register access
- Improved code readability
- Standardized structure

##  Output
LED toggling using CMSIS-based programming.
