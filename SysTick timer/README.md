# STM32 SysTick Timer

##  Description
This project demonstrates the use of the SysTick timer in STM32F401RE to generate precise delays and periodic interrupts.

##  Objective
To understand and implement time delays and periodic tasks using the built-in SysTick timer.

##  Components Used
- STM32F401RE
- LED

##  Software Used
- Proteus (for simulation)
- Code Composer Studio
- Embedded C

##  Working Principle
SysTick is a built-in timer in ARM Cortex-M processors. It generates an interrupt at regular intervals. By configuring the reload value, we can create accurate delays or periodic events.

##  Features
- Precise time delay generation
- Periodic interrupt handling
- No external timer required

##  Output
LED blinks at a fixed time interval using SysTick delay.
